# Lak-bay

Mobile Based Travel Recommendation and Travel Buddy Application

## About The Project

This project is for educational purpose only, this is made for Capstone 1 and 2

Built in Flutter and Dart

## Features
- Recommendation
- Travel Planner
- Emergency Tab
- Expense Planner
- Travel History
